# ClaudePet 番茄钟功能设计

## 概述

为 ClaudePet 桌面宠物添加轻量级番茄钟功能，复用现有的行为系统和气泡显示机制，让桌宠在用户专注工作时一起"认真工作"，休息时放松。

## 设计目标

- 复用现有代码，改动最小
- 与桌宠动画系统自然融合
- 气泡显示倒计时，不干扰用户工作
- 右键菜单控制，操作简单

---

## 第一节：番茄钟状态

### 状态定义

| 状态 | 动画 | 说明 |
|------|------|------|
| `idle` | 当前待机 | 番茄钟未启动 |
| `focus` | `coding` (认真工作) | 专注 25 分钟 |
| `break_time` | `sit` (放松) | 休息 5 分钟 |

### 状态转换

```
idle → focus → break_time → idle (循环)
```

- 用户点击"开始专注"进入 `focus`
- 25分钟结束自动进入 `break_time`
- 5分钟休息结束自动回到 `idle`
- 用户可随时放弃/暂停，回到 `idle`

## 第二节：计时器机制

### FocusTimer 类

在 `PetWindow` 中添加番茄钟计时器：

```python
class FocusTimer(QObject):
    tick = pyqtSignal(int, str)  # (剩余秒数, 状态)
    finished = pyqtSignal(str)   # 完成的状态 ("focus" 或 "break")

    FOCUS_DURATION = 1500  # 25分钟
    BREAK_DURATION = 300   # 5分钟

    def start_focus(self):
        """开始25分钟专注"""

    def start_break(self):
        """开始5分钟休息"""

    def pause(self):
        """暂停，保留剩余时间"""

    def resume(self):
        """恢复计时"""

    def stop(self):
        """停止计时，回到idle"""

    def _tick(self):
        """每秒更新，发射tick信号"""
```

### 计时逻辑

- 使用 QTimer，每1000ms触发一次
- 每秒更新 `_remaining` 并发射 `tick` 信号
- 时间到时发射 `finished` 信号，触发状态转换
- 暂停时记录剩余时间，恢复时继续

## 第四节：气泡显示

### 显示规则

| 状态 | 气泡内容 | 持续时间 |
|------|----------|----------|
| 专注中 | `🍅 24:59` | 每秒更新 |
| 休息中 | `☕ 4:59` | 每秒更新 |
| 专注完成 | `✅ 专注完成！` | 3秒 |
| 休息完成 | `💪 开始下一轮！` | 3秒 |

### 时间格式

- 分钟:秒数，如 `24:59`、`05:00`
- 不足10秒时显示 `0:09`

### 刷新机制

- `tick` 信号触发时调用 `show_bubble()`
- 使用 `show_bubble(text, duration=2000)` 每次显示2秒
- QTimer每秒触发，实现持续显示效果

## 交互流程

```
用户右键 → "开始专注"
    ↓
PetWindow.start_focus()
    ↓
FocusTimer.start_focus()
    ↓
动画切到 "coding"
气泡显示 "🍅 25:00"
    ↓
每秒 tick → 更新气泡 "🍅 24:59" → "🍅 24:58" → ...
    ↓
25分钟到 → finished 信号
    ↓
气泡显示 "✅ 专注完成！" (3秒)
动画切到 "sit"
FocusTimer.start_break()
    ↓
气泡显示 "☕ 5:00"
    ↓
每秒 tick → 更新气泡 "☕ 4:59" → ...
    ↓
5分钟到 → finished 信号
    ↓
气泡显示 "💪 开始下一轮！" (3秒)
动画回到 "idle"
```

---

## 边界情况

1. **暂停后恢复**：记录剩余时间，恢复时继续倒计时
2. **放弃专注**：停止计时器，动画回到 idle，气泡清空
3. **跳过休息**：停止计时器，直接回到 idle
4. **拖动桌宠**：番茄钟继续，气泡跟随移动
5. **最小化/隐藏**：番茄钟在后台继续计时
